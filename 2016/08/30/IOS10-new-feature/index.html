IOS10 new feature

发表于 2016-08-30   |     |   阅读次数

目的

  1. 全面了解IOS 10提供的新特性
  2. 了解哪些新特性可以引入到当前的业务以改进产品的用户体验

具体的实现细节不在本文档讨论范围之类

参考资料

  1. What’s New in iOS
  2. Best new features

支持设备列表

新特性

SiriKit

如果应用打算接入SiriKit，应用必须提供以下服务:

  • 音频&视频通讯
  • 信息（iMessage）
  • 付款和收款
  • 搜索照片
  • 预定行程
  • 管理运动数据

SiriK还提供了一个机制，当我们通过SiriKit向系统发起一个请求的时候，系统通过判断，看是否有app可以处理这些请求。
详情可以参考 SiriKit Programming Guide

Proactive Suggestions

增加应用的搜索特性，提高应用的推荐率，打开率，使用率。通过提供一些属性给系统，让系统更好的推荐app去完成特定的操作，比如预定酒店、视频&音频播放。该功能的实现设计到一系列的技术，比如：NSUserActivity, web markup defined by Schema.org, and APIs defined in the Core Spotlight, MapKit, UIKit, and Media Player frameworks.

Integrating with the Message App

通过集成Messages.framework，app可以方便实现通讯功能。
详情可以参考:

  • Messages Framework Reference
  • App Extension Programming Guide

User Notifications(通知&推送)

在锁屏模式下，用户可以直接在推送消息里面查看照片，观看视频，听音乐，甚至是直播.

  • 可以在收到Notification（local&remote）后，对Notification进行修改，然后再展示给用户
  • 可以自定义Notification UI及响应用户的操作

涉及的框架如下：

  • UserNotifications.framework
  • UserNotificationsUI.framework

Speech Recognition(语音识别)

  • 语音转文字。支持语音流和文件
  • 如果要识别日历和图片，需要获得用户的许可。苹果会把这些数据缓存到苹果的服务器上，以提高识别的准确度

涉及的框架如下：

  • Speech.framework

Wide Color

支持更加丰富的颜色显示

Adapting to the True Tone Display

根据外部的亮度情况自动调整设备的显示颜色和亮度

If your app is a reading app, conformance with the environmental white point is helpful to users. In this case, you can use the UIWhitePointAdaptivityStyleReading style to increase the strength of True Tone shift applied by the system.

App Search Enhancements

  • 应用内搜索
  • 搜索延长
  • 使用不同的隐私策略搜索速度链接
  • 有效结果可视化

Widget Enhancements

锁屏界面相关

Apple Pay Enhancements

苹果支付相关

Security and Privacy Enhancements

You must statically declare your app’s intended use of protected data classes by including the appropriate purpose string keys in your Info.plist file. For example, you must include the NSCalendarsUsageDescription key to access the user’s Calendar data. If you don’t include the relevant purpose string keys, your app exits when it tries to access the data.
